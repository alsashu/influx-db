Theory (20 minutes):
What is InfluxDB 3 Core?

Open-source time-series database built on Apache Arrow
Complete rewrite from InfluxDB 2.x
Designed for high-cardinality data and analytical workloads
Uses SQL for querying (major shift from InfluxQL/Flux)

Key Innovations:

Apache Arrow - Columnar in-memory format
Parquet - Persistent storage format
DataFusion - SQL query engine
Object Storage - Cloud-native architecture
