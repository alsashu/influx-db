How to Use
  • Draw Track:
    1. Click the "Draw Track" button (pencil icon turns blue, cursor becomes crosshair)
    2. Click on grid intersections to add points - each click adds a point to your track
    3. Continue clicking to build your track path with multiple points
    4. Double-click on your last point OR press Escape key to finish and create the track
    5. You can immediately start drawing another track by clicking again
  • Grid Size: Change the grid size (meters per square) to adjust measurement scale. All track lengths update automatically.
  • Select & Move: Click "Select" button, then click and drag tracks to reposition them. Points snap to grid intersections.
  • Delete: In Select mode, click a track to select it, then click Delete button or press Delete key.
  • Track Length: Each track displays its length in meters as a label. Length updates dynamically when moved or when grid size changes.
  • Export: Save your design as SVG (vector) or PNG (raster) image.
